นโยบายความเป็นส่วนตัว

แอปพลิเคชันนี้เป็นระบบภายในองค์กรของบริษัท ซึ่งใช้ข้อมูลตามความจำเป็นเพื่อดำเนินงานผ่าน Workflow ของ n8n และบันทึกผลลงใน Google Sheets

ข้อมูลที่เก็บและใช้งาน

ระบบอาจเข้าถึงข้อมูลดังต่อไปนี้:

  • ข้อมูลบัญชี Google เบื้องต้น (เช่น อีเมล)
  • สิทธิ์เข้าถึงหรือแก้ไข Google Sheets เฉพาะไฟล์ที่ใช้ภายในองค์กร

วัตถุประสงค์ของการใช้ข้อมูล

  • ใช้สำหรับรัน Workflow อัตโนมัติภายในบริษัท
  • บันทึกข้อมูลที่จำเป็นลงใน Google Sheets
  • ตรวจสอบสิทธิ์และยืนยันผู้ใช้งานในระบบ

การจัดเก็บและความปลอดภัย

ข้อมูลจะถูกจัดเก็บภายในระบบที่จำกัดการเข้าถึง และใช้เฉพาะภายในบริษัทเท่านั้น

การเผยแพร่ข้อมูล

ระบบ จะไม่เผยแพร่ ไม่ขาย และไม่แบ่งปันข้อมูลผู้ใช้งาน ให้บุคคลภายนอก

Privacy Policy

This application is an internal company system that processes data using n8n workflows and stores required results in Google Sheets.

Data Collected and Used

The system may access the following:

  • Basic Google account information (e.g., email)
  • Access to specific Google Sheets required for internal workflows

Purpose of Data Usage

  • Executing internal company workflows using n8n
  • Writing workflow results to Google Sheets
  • User authentication and verification

Data Storage and Security

All data is stored in controlled systems accessible only within the company.

Data Sharing

The system does not share, sell, or distribute any user data to external parties.
