ระบบ Internal Workflow Automation

แอปพลิเคชันนี้เป็นระบบอัตโนมัติภายในบริษัท ซึ่งใช้แพลตฟอร์ม n8n ในการรัน Workflow ต่างๆ และจัดเก็บข้อมูลลงใน Google Sheets เพื่อใช้สำหรับงานภายในองค์กร

วัตถุประสงค์ของแอป

  • เชื่อมต่อข้อมูลระหว่างระบบภายในบริษัท
  • ประมวลผลอัตโนมัติผ่าน Workflow ของ n8n
  • บันทึกข้อมูลลง Google Sheets เพื่อใช้ในการรายงานหรือการตรวจสอบ

ข้อมูลที่เข้าถึง

ระบบจะขอสิทธิ์เฉพาะที่จำเป็นสำหรับการทำงาน เช่น:

  • สิทธิ์เข้าถึง Google Sheets เพื่อเขียนข้อมูลลงในไฟล์
  • ข้อมูลบัญชีพื้นฐานของ Google เพื่อตรวจสอบผู้ใช้งาน

Internal Workflow Automation System

This application is an internal company automation system that uses the n8n platform to run workflows and stores processed data into Google Sheets for internal operations.

Purpose of the Application

  • Connect and synchronize internal company systems
  • Automate processes using n8n workflows
  • Store data into Google Sheets for reporting and auditing

Data Access Required

This application requests only the minimum access necessary for operation:

  • Access to Google Sheets for writing workflow data
  • Basic Google Account information for user verification
